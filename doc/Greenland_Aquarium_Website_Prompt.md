# Greenland Aquarium Website Design & Development Prompt

## Project Overview

Create a premium, award-winning website for **Greenland Aquarium**, a modern aquarium and pet store located in Bengaluru, India.

The design should feel like an **Awwwards-winning** creative website rather than a standard pet store. The experience should be cinematic, elegant, immersive, and animation-driven.
